# Character Consistency Guide

## 목적
- PC가 바뀌거나 툴이 달라져도 캐릭터 외형과 무드가 크게 흔들리지 않게 하기 위한 운영 문서
- 캐릭터 생성은 항상 `고정값 + 상황값`으로 나눠서 진행
- 고정값은 절대 바꾸지 않고, 상황값만 바꿔서 cover / thumb / 숏폼 컷을 확장

## 공통 운영 원칙
- 같은 캐릭터는 가능한 한 같은 모델 계열과 같은 톤의 프롬프트 구조를 유지한다.
- 생성 전에 반드시 캐릭터 시트의 `비주얼 방향`, `스타일 방향`, `이미지 제작 가이드`, `현재 확정 요소`를 먼저 읽는다.
- cover는 전신 또는 전신이 명확히 읽히는 구도로 만든다.
- thumb는 얼굴 중심 클로즈업 또는 어깨 위 상반신 중심으로 만든다.
- 생성 후에는 아래 5가지를 체크한다.
  - 얼굴형이 맞는가
  - 헤어가 맞는가
  - 나이대 느낌이 맞는가
  - 캐릭터 무드가 맞는가
  - 금지 요소가 들어가지 않았는가

## 프롬프트 구조
- 고정값: 캐릭터 정체성, 얼굴, 헤어, 체형, 무드, 금지 요소
- 상황값: 포즈, 배경, 소품, 표정, 카메라 거리

### 기본 템플릿
```text
캐릭터 고정값:
- 국적
- 나이대 느낌
- 얼굴형
- 눈매 / 표정
- 피부 표현
- 헤어
- 체형
- 전체 인상
- 메인 스타일
- 금지 요소

이번 컷 상황값:
- 용도: cover 또는 thumb
- 포즈
- 배경
- 소품
- 표정
- 구도

출력 조건:
- 한 명만
- 워터마크 없음
- 글자 없음
- 손/팔다리 왜곡 없음
- 캐릭터 시트의 금지 요소 반영
```

## 캐릭터 기준 샷
- 각 캐릭터마다 최소 4장을 기준 샷으로 유지한다.
  - cover 1장
  - thumb 1장
  - 정면 기준 샷 1장
  - 분위기 확장 샷 1장
- 새 컷을 만들 때는 기존 성공 이미지와 캐릭터 시트를 함께 보고 만든다.

## 최서진

### 고정값
- 포지션: 프리미엄 메인 / 배우 라인 / 브랜드 간판
- 레퍼런스 결: 김혜수 결
- 국적: 한국인
- 나이대 느낌: 30대 초중반
- 얼굴형: 선이 또렷하고 안정감 있는 귀족적인 얼굴선
- 눈매: 날카롭지만 과하게 차갑지 않은 깊은 눈빛
- 입술 / 표정: 도톰한 입술, 여유롭고 관능적인 미소
- 피부: 매끈하고 윤기 있는 하이엔드 물광 피부
- 헤어: 턱선까지 오는 정돈된 스트레이트 연갈색 보브컷
- 체형: 성숙하고 자연스러운 글래머러스 hourglass silhouette
- 전체 인상: 관록과 여유가 묻어나는 우아함, 최정상급 배우와 하이엔드 모델의 중간 지점

### 메인 무드
- 카리스마
- 고급감
- 존재감
- 성숙한 지성미
- 압도적인 톱스타 무드

### 메인 스타일
- 블랙 / 샴페인 골드 / 딥 브라운
- 벨벳 / 실크 / 새틴 / 구조적인 럭셔리 드레스
- 절제된 럭셔리
- 배우 / 브랜드 캠페인 / 에디토리얼 무드

### 금지 요소
- 하이틴 무드
- 과한 핑크 / 러블리
- 귀여운 액세서리
- 경박한 섹시함
- 아이돌 느낌

### cover 기준
- 전신이 분명히 보이는 컷
- 레드카펫 / 시상식 / 럭셔리 계단 배경 가능
- 뒤돌아서 시선을 주는 포즈 가능
- 블랙 / 골드 드레스 중심
- 배우 / 모델 / 브랜드 간판 느낌 유지

### thumb 기준
- 얼굴 중심 클로즈업 또는 어깨 위 상반신
- 눈빛과 피부 표현이 강하게 보여야 함
- 배우 프로필 / 브랜드 헤드샷처럼 안정감 있는 구도

### 현재 프로젝트 기준 파일
- `assets/characters/choi-seojin/cover.png`
- `assets/characters/choi-seojin/thumb.png`

## 박도아

### 고정값
- 포지션: 메인 3인 / 스트리머 라인 / 커뮤니티 전환형 캐릭터
- 레퍼런스 결: 쯔양 결
- 국적: 한국인
- 나이대 느낌: 20대 초중반
- 얼굴형: 동글동글하고 귀여운 햄스터상, 웃을 때 올라오는 사랑스러운 볼살
- 눈매: 똘망똘망하고 맑은 눈망울, 활짝 웃을 때 반달 모양의 선한 눈웃음
- 피부: 복숭아처럼 맑고 혈색 도는 건강한 피부
- 헤어: 풀뱅 앞머리가 있는 자연스럽고 찰랑거리는 긴 다크 브라운 생머리
- 체형: 아담하고 슬림하지만 밝은 활력과 생기가 느껴지는 실루엣
- 전체 인상: 옆집 동생처럼 친근하고 보고 있으면 기분이 좋아지는 인간 비타민

### 메인 무드
- 친근함
- 밝음
- 생활감
- 리액션
- 건강한 에너지

### 메인 스타일
- 캐주얼 스트리머룩
- 후드 / 니트 / 편한 탑 / 데님 / 스포티 아이템
- 생활감 있는 컬러
- 오렌지 / 코랄 / 따뜻한 뉴트럴

### 금지 요소
- 차갑고 하이패션스러운 연출
- 지나치게 고급 화보형 거리감
- 강한 섹시 콘셉트
- 과한 무대형 아이돌 스타일

### cover 기준
- 전신이 명확히 보여야 함
- 크리에이터 컨셉 유지
- 의자에 앉아서 먹방 하기 직전의 장면 가능
- 음식 / 간식 / 음료 / 방송 셋업 포함 가능
- 밝은 리액션과 생활감이 바로 읽혀야 함

### thumb 기준
- 표정 큰 얼굴 중심 클로즈업
- 밝은 미소 / 놀람 / 먹방 리액션
- SNS 썸네일에서 바로 눈에 띄는 구도

### 현재 프로젝트 기준 파일
- `assets/characters/park-doa/cover.png`
- `assets/characters/park-doa/thumb.png`

## 다른 PC에서 다른 캐릭터 2명을 만들 때 지시하는 방법
- 먼저 해당 PC의 Codex 또는 사용하는 이미지 툴에 아래 순서대로 지시한다.

### 1. 작업 순서 지시
```text
이 캐릭터는 먼저 캐릭터 시트를 읽고 작업해줘.
반드시 비주얼 방향, 스타일 방향, 이미지 제작 가이드, 현재 확정 요소를 먼저 반영해.
cover를 먼저 만들고, cover가 확정되면 그 다음에 thumb를 만들어.
cover는 전신 기준으로 만들고, thumb는 얼굴 중심 클로즈업으로 만들어.
캐릭터 고정값은 유지하고 포즈, 배경, 소품만 바꿔.
```

### 2. 캐릭터 고정값 추출 지시
```text
이 캐릭터의 고정값만 먼저 정리해줘.
정리 항목은 국적, 나이대 느낌, 얼굴형, 눈매, 피부 표현, 헤어, 체형, 전체 인상, 메인 스타일, 금지 요소야.
이 고정값은 이후 cover와 thumb에서 절대 바꾸지 마.
```

### 3. cover 생성 지시
```text
이 캐릭터의 cover를 만들어줘.
조건:
- 전신이 분명히 보여야 해
- 캐릭터 시트의 이미지 제작 가이드 기준
- 캐릭터 고정값 유지
- 상황값만 이번 컷에 맞게 적용
- 글자 없음
- 워터마크 없음
- 한 명만
- 금지 요소 반영

먼저 1장을 만들고,
얼굴 / 헤어 / 무드 / 체형 / 의상 / 배경이 시트와 맞는지 스스로 체크해줘.
```

### 4. thumb 생성 지시
```text
이 캐릭터의 thumb를 만들어줘.
조건:
- 얼굴 중심 클로즈업 또는 어깨 위 상반신
- 캐릭터 시트의 thumb 방향 기준
- cover와 같은 캐릭터로 보여야 함
- 헤어, 얼굴형, 피부, 무드는 유지
- 글자 없음
- 워터마크 없음
- 한 명만
```

### 5. 최종 검수 지시
```text
최종 검수해줘.
아래 항목으로 체크:
- 같은 캐릭터로 보이는가
- cover는 전신 기준을 만족하는가
- thumb는 얼굴 중심인가
- 헤어가 시트와 일치하는가
- 나이대 느낌이 맞는가
- 금지 요소가 들어가지 않았는가
```

## 다른 PC용 짧은 명령 예시
```text
이 캐릭터는 캐릭터 시트를 먼저 읽고 작업해.
고정값은 유지하고 상황값만 바꿔.
cover부터 만들고, cover 확정 후 thumb를 만들어.
cover는 전신, thumb는 얼굴 중심 클로즈업.
반드시 같은 캐릭터로 보이게 유지하고 금지 요소는 넣지 마.
```

## 운영 팁
- 다른 PC에서 작업할 때는 가능하면 이 문서와 캐릭터 시트를 같이 열어두고 시작한다.
- 새 이미지가 잘 나오면 그 즉시 `cover` / `thumb`를 저장하고 Git에 올린다.
- 같은 캐릭터를 다른 모델에서 반복 생성할수록 흔들릴 수 있으니, 이미 잘 나온 결과는 기준샷으로 계속 참조한다.
