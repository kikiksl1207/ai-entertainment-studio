# Roadmap

## Phase 1. 사업 구조 정리
- 캐릭터 10개 기획
- 역할 기반 운영 구조 수립
- 사이트 메뉴 구조 확정

## Phase 2. 콘텐츠 테스트
- 메인 캐릭터 3개 선정
- 숏폼 발행
- 반응 데이터 수집

## Phase 3. 팬 플랫폼 오픈
- 캐릭터 상세 페이지
- 후원 기능
- 랭킹/활동 구조

## Phase 4. 수익화 강화
- 광고/제휴 메뉴 오픈
- 화보 판매
- 멤버십 실험

## Phase 5. 대화형 확장
- 캐릭터챗 MVP
- 캐릭터별 말투/기억/관계도 설계

## Phase 6. 추후 확장
- 반실시간 이벤트
- 라이브 실험
- 다국어 채널 확장
