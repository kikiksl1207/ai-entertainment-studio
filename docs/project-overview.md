# Project Overview

## 서비스 정의
ALLI Entertainment Studio는 AI 캐릭터를 제작하고, 캐릭터별 숏폼 콘텐츠를 지속적으로 발행하며, 팬 후원과 광고 및 유료 상호작용으로 수익화하는 AI 엔터테인먼트 사업이다.

## 핵심 구성
- 캐릭터 IP 제작
- 캐릭터별 프로필/세계관 관리
- 숏폼/릴스 자동 생산
- 팬 후원 및 랭킹 구조
- 화보/브랜드 제휴
- 추후 캐릭터챗 확장

## 1차 목표
- 캐릭터 10개 세팅
- 메인 운영 캐릭터 3개 선정
- 캐릭터 소개 사이트 오픈
- 캐릭터별 콘텐츠 피드 구성
- 후원 및 캐릭터 업그레이드 구조 설계

## 제외 범위
- 실시간 방송
- 다중 언어 확장
- 오프라인 굿즈 운영

## 성공 지표
- 캐릭터별 팔로워 증가율
- 숏폼 평균 조회수
- 팬 후원 전환율
- 캐릭터별 재방문율
- 광고 제휴 문의 수
