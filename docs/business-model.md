# Business Model

## 수익 축

### 1. 팬 후원
- 캐릭터별 직접 후원
- 목표 달성형 후원
- 특정 의상/화보/신규 콘텐츠 해금형 후원

### 2. 광고
- 브랜드 숏폼 협업
- 화보형 광고
- PPL형 캐릭터 노출

### 3. 유료 콘텐츠
- 멤버십 전용 화보
- 고화질 세트
- 비하인드 / 미공개 콘텐츠

### 4. 캐릭터챗
- 월 구독
- 회차 과금
- 관계도 기반 유료 상호작용

### 5. 확장 수익
- 디지털 굿즈
- 포토카드
- 시즌 한정 테마팩

## 후원 사용 방식
- 캐릭터 외형 업그레이드
- 숏폼 발행 빈도 증가
- 신규 콘셉트 공개
- 팬 전용 이벤트 해금

## 운영 우선순위
1. 팬 후원
2. 광고
3. 유료 화보
4. 캐릭터챗
