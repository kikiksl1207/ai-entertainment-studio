# Han Seoyul Asset Pack

## 기준

- 캐릭터명: 한서율 (Han Seoyul)
- 기준 이미지: `cover.png`
- 핵심 인상: 럭셔리 러블리 메인 센터, 밝고 화사한 한국 아이돌 비주얼
- 얼굴 기준: 작은 얼굴, 짧은 중안부, 갸름한 V라인, 큰 둥근 눈과 도톰한 애교살, 복숭아빛 물광 피부, 도톰한 글로시 핑크 립
- 헤어/스타일: 긴 다크 브라운 S컬 웨이브, 샴페인/블러시 핑크 크리스털 무대룩, 진주/비즈/시폰 디테일
- 색감: blush pink, champagne gold, pearl white, peach coral

## 선택

- `cover.png`: 사용자가 제공한 커버 기준 얼굴로 고정
- `thumb.png`: 커버와 동일한 얼굴 기준의 썸네일 이미지
- `reference-final-01.png` - `reference-final-10.png`: 기존 한서율 컷 중 얼굴/의상/무대 무드가 가장 안정적인 컷
- `reference-final-11.png` - `reference-final-20.png`: 2026-05-02 신규 생성 후보 10장 중 최종 채택 컷

## 주의사항

- `seo-yuan`은 별도 캐릭터이므로 한서율 검수/정리 대상이 아니다.
- 다른 캐릭터 얼굴과 섞이지 않도록 `cover.png`의 얼굴형, 눈매, 글로시 립, 핑크 크리스털 무대룩을 기준으로 비교한다.
- 차갑고 시크한 냉미녀 무드, 긴 중안부, 얇고 건조한 입술, 교복/스트리머/일상 브이로그 무드는 한서율 기준에서 제외한다.
- 기존 `reference/` 폴더는 레거시 후보와 보류 근거로 보존한다.
