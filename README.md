# ALLI Entertainment Studio

이 프로젝트는 기존 `alli-mvp`와 분리된 별도 사업/서비스 트랙입니다.

목표:
- AI 캐릭터 IP를 제작하고 관리한다.
- 캐릭터별 숏폼/릴스/롱폼 파이프라인을 운영한다.
- 팬 후원, 광고, 화보, 대화형 캐릭터챗으로 수익화한다.
- 향후 실시간 방송 기능은 별도 단계로 확장한다.

이 폴더는 `alli-mvp`의 제품 MVP와 섞이지 않도록 다음 범위만 다룹니다.
- 사업 구조
- 사이트 구조
- 캐릭터 운영 구조
- 역할 분리형 AI 에이전트 정의
- 추후 개발 범위 정리

폴더 구조:
- `docs/` : 사업 기획, 사이트 구조, 로드맵
- `agents/` : 역할 기반 AI 에이전트 정의

초기 운영 원칙:
1. 캐릭터 10개를 모두 동등하게 운영하지 않는다.
2. 메인 캐릭터 2~3개에 우선 집중한다.
3. 사이트는 팬 관리와 수익화 중심으로 설계한다.
4. 선정성 가능성이 있으므로 초기부터 정책 리스크를 고려한다.
