# AI Entertainment Studio

이 프로젝트는 기존 `alli-mvp`와 분리된 별도 엔터/캐릭터 플랫폼 트랙입니다.

## 현재 포함 범위
- 사업/운영 문서
- 역할 분리형 에이전트 정의
- 엔터사 톤의 정적 홈페이지 프로토타입

## 파일 구조
- `index.html` : 첫 랜딩/홈페이지 프로토타입
- `styles.css` : 공통 스타일 및 모바일 반응형
- `app.js` : 메인 3인, 숏폼, 라인업 더미 데이터 렌더링
- `assets/` : 실제 캐릭터 이미지 자산 위치
- `docs/` : 사업 구조, 사이트 구조, 로드맵
- `agents/` : 역할 기반 AI 에이전트 정의

## 실행 방법
설치 없이 바로 브라우저에서 `index.html`을 열면 됩니다.

## 초기 운영 원칙
1. 캐릭터 10개를 모두 동등하게 운영하지 않는다.
2. 메인 캐릭터 3인에 우선 집중한다.
3. 사이트는 팬 유입과 브랜드 신뢰를 동시에 보여줘야 한다.
4. 모바일에서도 메인 탐색과 숏폼 소비가 자연스럽게 이어지도록 설계한다.

## 이미지 자산 넣는 법
- 자산 가이드는 `assets/README.md` 참고
- 메인 3인 + 프리미엄 메인부터 먼저 넣는 것을 권장
- 이미지가 없으면 현재는 그라데이션 카드로 fallback 됩니다
