# Agent Roles

이 폴더는 AI 에이전트 또는 실제 팀 역할을 분리하기 위한 정의서입니다.

핵심 원칙:
- 한 에이전트가 모든 결정을 하지 않는다.
- 전략, 수익, 콘텐츠, 캐릭터, 팬 운영, 제품을 분리한다.
- 결과물은 서로 주고받되 책임 범위는 명확히 둔다.
